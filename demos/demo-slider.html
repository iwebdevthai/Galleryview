GalleryView » Image Slider

To create a slider style gallery, we need only make two changes to the default options: disable the filmstrip, and change the panel animation style. For this demo, I have also disabled the info bar and enabled a theme with larger navigation buttons.

$('#myGallery').galleryView({
    transition_speed: 600,
    show_filmstrip: false,
    panel_animation: 'slide',
    show_infobar: false
});
